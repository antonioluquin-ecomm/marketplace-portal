Gestión de Sellers

Usá esta pantalla para crear o actualizar la ficha inicial de un seller y dejarla disponible para el Backlog, los formularios públicos y el seguimiento operativo.

El ID del seller se genera automáticamente. Las fechas de alta y actualización las completa el sistema. Los logos, la vista previa y los links públicos se calculan a partir del ID.

Cómo usar esta pantalla

Para un seller nuevo, completá solo la información inicial disponible. Si el seller ya existe, seleccionalo para editar su ficha sin cambiar el ID. La calificación y el relevamiento completan luego los datos técnicos faltantes.
