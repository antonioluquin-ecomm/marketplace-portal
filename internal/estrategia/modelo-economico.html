Modelo Económico

Esta página define cómo se analiza la viabilidad económica de un seller dentro del Marketplace Sporting. Su objetivo es ordenar comisiones, costos, servicios, liquidación y uso de simuladores, separando con claridad qué información es interna y qué información puede compartirse con sellers.

Propósito

Establecer un marco común para evaluar si la incorporación de un seller es rentable, sostenible y coherente con el modelo operativo definido para Sporting.

La comisión no debe analizarse de forma aislada. Cada caso debe contemplar financiación, logística, operación, devoluciones, conciliación y nivel de servicio asumido por Sporting.

Uso de la página

Interno: guía para evaluación comercial, técnica y operativa.

No reemplaza: simuladores, contratos ni acuerdos comerciales finales.

Conecta con: onboarding, gobernanza, Backlog y Simulador Económico.

Regla económica principal

Toda evaluación debe partir del neto económico real, no solo de la venta bruta.

Estructura de análisis
Venta cobrada - Comisión - Costos financieros - Costos operativos = Neto seller / margen Sporting

Esta fórmula funciona como marco conceptual. Los valores concretos pueden cambiar según el acuerdo, pero la estructura de análisis debe mantenerse consistente para todos los sellers.

Regla de control

Si un seller requiere excepciones comerciales, logísticas o financieras que alteran el modelo estándar, el caso debe pasar por gobernanza antes de avanzar en planificación o integración.

Lógica de liquidación

La liquidación debe estar asociada a pedidos válidos y cobrados, contemplando excepciones operativas.

01

Pedido cobrado

Venta efectivamente cobrada dentro del período.

02

Depuración

Se excluyen cancelados, no entregados o casos no liquidables.

03

Deducciones

Se aplican comisión, costos financieros, servicios y costos operativos.

04

Conciliación

Se validan diferencias, devoluciones, notas de crédito y reclamos.

05

Liquidación

Se determina el neto a pagar al seller.

Criterio recomendado

Trabajar sobre pedidos cobrados, descontando cancelaciones, no entregados, devoluciones y conceptos económicos acordados. Esto permite una conciliación más clara y trazable.

Regla de confidencialidad

El simulador interno no debe compartirse con sellers si contiene costos reales, márgenes internos, criterios de rentabilidad o parámetros sensibles. Para uso externo debe existir una versión simplificada.

Criterios para avanzar

Antes de priorizar un seller, el acuerdo económico debe pasar estos controles mínimos.

CriterioQué se validaResultado esperado

CompetitividadEl precio final resulta atractivo para el cliente y el mercado.Competitivo

ConciliaciónExisten reglas para liquidaciones, diferencias y devoluciones.Trazable

Conclusión institucional

El modelo económico funciona como marco de decisión y control. Un seller no debe avanzar solo por potencial comercial: también debe demostrar viabilidad económica, operativa y de conciliación.
